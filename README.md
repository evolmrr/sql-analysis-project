# SQL Data Analysis Project

## 📊 Project Overview
This project focuses on analyzing a relational database using SQL to solve real-world business problems.  
The analysis includes customer behavior, marketing efficiency, product relationships, and revenue dynamics.

---

## 🎯 Business Tasks

### 1. Marketing Efficiency
Determine the day when revenue from customers acquired through a marketing campaign exceeded the cost of acquisition.

### 2. Product Analysis
Identify which pairs of products are most frequently purchased together.

### 3. Revenue Analysis
Calculate daily revenue and daily revenue growth:
- absolute growth  
- percentage growth compared to the previous day  

---

## 📁 Project Structure

- `basic.sql`  
  Analysis of marketing campaign performance, including revenue vs acquisition cost and supporting dashboards  

- `join.sql`  
  Product analysis using JOIN operations to find frequently co-purchased items  

- `window_functions.sql`  
  Revenue analysis using window functions to calculate daily revenue and growth metrics  

---
